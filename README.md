# Project Description
I’ve been working on a project that turns a foam dart blaster into a fully autonomous sentry turret using YOLO11. Basically, the AI acts as the "brain," constantly scanning the room to find and lock onto targets in real-time. Once it spots something, it automatically tracks the movement and lets darts fly. It’s been an awesome challenge getting the software to talk to the hardware fast enough to actually hit a moving target.

Right now, it’s focused on the core "track and fire" loop, but I’m just getting started. I’m planning to add some cooler features down the road, like predictive aiming so it can lead targets and maybe even some basic facial recognition so it doesn't accidentally blast my friends. It’s a fun mix of computer vision and DIY engineering, and it’s honestly just satisfying to see it hunt for targets on its own.
